# AI-Powered Attendance Intelligence Platform

## Overview

This project demonstrates how Microsoft technologies can be combined to automate attendance monitoring, reporting, analytics and decision support.

## Technologies Used

- Microsoft Excel
- Microsoft Copilot
- Power BI
- Power Automate
- Copilot Studio
- GitHub

## Business Problem

Educational institutions often rely on manual attendance tracking and reporting.

This project demonstrates how AI-powered insights and automation can improve decision-making and reduce manual effort.

## Solution Architecture

Excel Dataset -> Copilot Analysis -> Power BI Dashboard -> Power Automate Alerts -> Copilot Studio Assistant

## Features

- Attendance analytics
- At-risk student identification
- Automated reporting
- AI-generated insights
- Conversational assistant design

## Future Enhancements

- Azure AI Foundry Integration
- Dataverse Integration
- Teams Notifications
